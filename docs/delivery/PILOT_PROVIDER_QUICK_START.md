# PILOT_PROVIDER_QUICK_START.md

## Guia rapida para proveedores

Gracias por participar como proveedor piloto de Pet Ecosystem. Esta prueba valida la configuracion del negocio, reservas y operacion del servicio.

Durante el piloto no se realizaran cobros reales.

## 1. Instalar la app

1. Abre el enlace privado que recibiste.
2. Descarga el APK.
3. Toca el archivo descargado.
4. Si Android lo solicita, permite instalar apps desde esa fuente.
5. Instala y abre `Pet Ecosystem`.

## 2. Crear tu cuenta

1. Selecciona `Crear cuenta`.
2. Escribe email, contrasena, nombre y apellido.
3. Elige el rol `Proveedor`.
4. Crea la cuenta.
5. Verifica tu correo si la app lo solicita.
6. Inicia sesion.

## 3. Completar perfil y negocio

1. Entra a `Cuenta`.
2. Completa tus datos personales.
3. Entra a `Negocio`.
4. Crea tu negocio.
5. Revisa que el nombre del negocio sea claro para los propietarios.

## 4. Perfil publico

1. Entra a `Negocio`.
2. Abre `Perfil publico`.
3. Completa la presentacion visible para propietarios.
4. Sube logo/foto si esta disponible.
5. Marca el perfil como publico si corresponde.

## 5. Ubicacion publica

1. Entra a `Negocio`.
2. Abre `Ubicacion publica`.
3. Completa ciudad, pais y nombre visible.
4. Usa coordenadas aproximadas desde la ciudad si aplica.
5. Confirma que la ubicacion este publica.

## 6. Crear servicio

1. Entra a `Servicios`.
2. Toca el icono de agregar.
3. Crea el servicio asignado para el piloto.
4. Define precio referencial y duracion.
5. Marca el servicio como activo y publico.

## 7. Crear horarios/capacidad

1. Entra a `Horarios`.
2. Crea una regla de disponibilidad para el servicio.
3. Selecciona dia, hora inicio, hora fin y capacidad.
4. Guarda.
5. Confirma que el horario aparezca como activo.

## 8. Documentos y aprobacion

1. Entra a `Estado`.
2. Revisa pasos pendientes.
3. Sube documento de aprobacion si se solicita.
4. Espera aprobacion del admin.
5. Cuando el admin apruebe, confirma que tu negocio aparezca en marketplace.

## 9. Gestionar reserva

1. Entra a `Reservas`.
2. Revisa reservas pendientes.
3. Abre la reserva.
4. Aprueba si corresponde.
5. Mantente atento a reservas confirmadas.

## 10. Escanear QR

1. En la reserva confirmada, abre el detalle operativo.
2. Toca la accion de escanear QR.
3. Escanea el QR de check-in que muestra el propietario.
4. Al finalizar, escanea el QR de check-out.
5. Revisa que el timeline muestre entrada y salida registradas.

## 11. Subir evidencia documental

1. Despues del check-out, busca `Cargar evidencia de actividad`.
2. Selecciona el documento o imagen.
3. Carga la evidencia.
4. Confirma que el timeline muestre `Documento registrado`.

## 12. Reportar problemas

Si algo falla:

1. Toma captura de pantalla.
2. Anota el paso exacto.
3. Indica modelo del telefono y version Android si la sabes.
4. Envia el reporte por el canal de soporte del piloto.

Ejemplo:

```text
Soy PROV-01. Estaba escaneando QR de check-in. Error: [texto]. Telefono: [modelo]. Adjunto captura.
```
