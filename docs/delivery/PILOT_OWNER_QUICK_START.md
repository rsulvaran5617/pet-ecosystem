# PILOT_OWNER_QUICK_START.md

## Guia rapida para propietarios

Gracias por participar en el piloto de Pet Ecosystem. Esta prueba ayuda a validar el flujo real de reserva y servicio para mascotas.

Durante el piloto no se realizaran cobros reales.

## 1. Instalar la app

1. Abre el enlace privado que recibiste.
2. Descarga el APK.
3. Toca el archivo descargado.
4. Si Android lo solicita, permite instalar apps desde esa fuente.
5. Instala y abre `Pet Ecosystem`.

## 2. Crear tu cuenta

1. Selecciona `Crear cuenta`.
2. Escribe email, contrasena, nombre y apellido.
3. Elige el rol `Propietario de mascota`.
4. Crea la cuenta.
5. Verifica tu correo si la app lo solicita.
6. Inicia sesion.

## 3. Completar tu perfil

1. Entra a `Cuenta`.
2. Completa tus datos personales.
3. Agrega una direccion si aplica.
4. Confirma que el modo activo sea propietario.

## 4. Crear hogar y mascota

1. Entra a `Mascotas`.
2. Crea o confirma tu hogar.
3. Registra tu mascota.
4. Agrega foto/avatar si quieres.
5. Revisa que la mascota este activa.

## 5. Buscar proveedor

1. Entra a `Buscar`.
2. Busca el proveedor asignado para tu prueba.
3. Abre el perfil del proveedor.
4. Revisa servicio, ubicacion y disponibilidad.

## 6. Crear reserva

1. Selecciona el servicio.
2. Toca `Ver horarios con cupo` si aparece.
3. Elige fecha y horario disponible.
4. Genera la vista previa.
5. Revisa mascota, servicio, fecha y horario.
6. Confirma la reserva.

Nota: generar vista previa no consume cupo. El cupo se toma al confirmar la reserva.

## 7. Usar QR durante el servicio

1. Abre la reserva confirmada.
2. Al inicio del servicio, muestra el QR de check-in al proveedor.
3. Espera que el proveedor lo escanee.
4. Al finalizar, muestra el QR de check-out.
5. Espera que el proveedor lo escanee.

## 8. Revisar timeline y evidencia

1. Entra al detalle de la reserva.
2. Revisa que aparezca check-in registrado.
3. Revisa que aparezca check-out registrado.
4. Revisa si aparece evidencia documental o actividad registrada.

## 9. Reportar problemas

Si algo falla:

1. Toma captura de pantalla.
2. Anota que estabas intentando hacer.
3. Indica modelo del telefono y version Android si la sabes.
4. Envia el reporte por el canal de soporte del piloto.

Ejemplo:

```text
Soy OWN-01. Estaba intentando confirmar reserva. Error: [texto]. Telefono: [modelo]. Adjunto captura.
```
