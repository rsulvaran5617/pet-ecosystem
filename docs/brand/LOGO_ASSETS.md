# Pet Ecosystem logo assets

## Fuente

El set inicial se genero desde `C:\Users\Ramon Sulvaran\Downloads\PetIcons.png`, un PNG cuadrado de `1254x1254` sin transparencia.

## Assets web

- `apps/web/public/brand/pet-ecosystem-logo-source.png`: copia de referencia del arte recibido.
- `apps/web/public/brand/pet-ecosystem-logo-horizontal.png`: logo horizontal con fondo transparente para piezas amplias.
- `apps/web/public/brand/pet-ecosystem-logo-mark.png`: isotipo transparente para navegacion, tarjetas y UI compacta.
- `apps/web/public/brand/pet-ecosystem-mark-1024.png`: isotipo transparente centrado en lienzo cuadrado.
- `apps/web/public/brand/icon-192.png`: icono web.
- `apps/web/public/brand/icon-512.png`: icono web/PWA.
- `apps/web/public/brand/icon-1024.png`: icono maestro cuadrado.
- `apps/web/public/brand/apple-touch-icon.png`: icono iOS web.

## Assets mobile

- `apps/mobile/assets/brand/pet-ecosystem-logo-mark.png`: isotipo para splash y UI.
- `apps/mobile/assets/brand/pet-ecosystem-app-icon.png`: icono principal Expo.
- `apps/mobile/assets/brand/pet-ecosystem-adaptive-foreground.png`: foreground Android adaptive icon.
- `apps/mobile/assets/brand/pet-ecosystem-adaptive-background.png`: fondo solido de referencia.

## Uso

- Usar el isotipo en espacios pequenos.
- Usar el logo horizontal solo donde haya ancho suficiente.
- No usar el PNG compuesto original como icono, porque contiene multiples piezas y fondo no transparente.
